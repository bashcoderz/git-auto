
DESCRIPTION:
------------
Git Auto is a tool that helps programmers and coders automate Git operations with ease, without the trouble of remembering complex commands.

It includes awesome features like git stash ( to temporarily save your work without committing, allowing you to handle urgent tasks first ) and git grep ( which helps you find patterns in your project with ease).

How run the project:
--------------------
1.Download the script file.

2.Locate the folder where the script was downloaded (usually the Downloads folder).

3.Enable execution permission using the command inside quotation marks: 'sudo chmod +x git-auto.sh'

4.Run the script using the command inside quotation marks: 'bash git-auto.sh' Once executed, the git-auto logo will appear, indicating that the script is running.

5.Feel free to explore the project and test its different functionalities.
